# Bajaj Finserv Health Dev Challenge - Qualifier 1
1. Clone the repository:
```bash
git clone <https://github.com/SUPREME-AKASH-DEVELOPER/BajajFinservProjectt.git>
```

2. Install dependencies:
```bash
# Backend
cd backend
npm install

# Frontend
cd frontend
npm install
```

3. Start the development servers:
```bash
# Backend
npm run dev

# Frontend
npm run dev
```
